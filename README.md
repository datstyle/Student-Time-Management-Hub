# Student Time Management Hub

## Project Title

Student Time Management Hub

## Project Description

Student Time Management Hub is a decentralized smart contract platform to manage student tasks, schedules, and study sessions with transparent progress tracking. Built using Soroban on the Stellar blockchain, it provides trustless management of academic deadlines, study time recording, and performance statistics — all enforced on-chain without relying on any centralized intermediary.

## Project Vision

The vision of Student Time Management Hub is to offer students a decentralized, secure, and transparent way to manage their academic time and tasks without relying on centralized platforms. By recording every task, schedule, and study session directly on the Stellar blockchain, we ensure tamper-proof academic records and verifiable study progress — empowering students with full ownership of their learning journey.

## Key Features

- **Task Management:** Students create and manage academic tasks with title, subject, priority level, deadline, and estimated study hours.
- **Priority System:** Tasks support four priority levels — Low, Medium, High, and Critical — for effective workload planning.
- **Status Tracking:** Task statuses (Todo, InProgress, Completed, Overdue) are updated on-chain with automatic overdue detection.
- **Schedule Management:** Students register class schedules, exam slots, and recurring events with location and time information.
- **Study Session Recording:** Students start and end study sessions on-chain with automatic duration calculation in minutes.
- **Performance Statistics:** On-chain student stats track total tasks, completed tasks, overdue tasks, total study minutes, and session count.
- **Completion Rate:** A public completion rate function calculates and returns each student's task completion percentage (0–100%).
- **Access Control:** Admin-restricted global management and student-restricted personal task and session control.
- **Immutable Records:** All tasks, schedules, and study sessions are permanently recorded on-chain for full auditability.
- **Transparent Status:** Publicly accessible student statistics and task history for accountability and verification.

## Usage Instructions

1. **Set Admin:** Deploy the contract and assign an admin address for global management.
2. **Create Tasks:** Students create academic tasks with subject, priority, deadline, and estimated study hours.
3. **Update Status:** Students update task status as they progress — Todo, InProgress, Completed, or Overdue.
4. **Create Schedules:** Students register class schedules, exam timetables, and recurring events with location data.
5. **Track Study Sessions:** Students start a study session on-chain and end it when done — duration is automatically calculated.
6. **Query Stats:** Anyone can query a student's completion rate, total study hours, task history, and session records for full transparency.

## Future Scope

- **Payment Integration:** Integrate Stellar token contracts (SEP-41) to reward students with tokens upon task completion or study milestones.
- **Group Study Sessions:** Support collaborative on-chain study sessions for student groups and project teams.
- **GPA Tracking:** Extend the system to link academic tasks with grades and GPA calculation on-chain.
- **Leaderboard System:** Add a public on-chain leaderboard ranking students by completion rate and study hours.
- **User Dashboards:** Build frontend interfaces for students to visualize their schedules, tasks, and study progress in real time.
- **Notification System:** Add on-chain or off-chain alerts for approaching deadlines, overdue tasks, and session reminders.
- **Institution Integration:** Allow universities to verify student study records and task completion on-chain.
- **Compliance Tools:** Automate academic reporting for scholarship eligibility and regulatory compliance.

## Technology Stack

- Rust and Soroban SDK for secure and performant smart contract development.
- Stellar blockchain for decentralized, low-cost, and immutable state management.
- Cryptographic signing and on-chain timestamping for secure task and session enforcement.
